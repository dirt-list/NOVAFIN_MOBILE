# Manual de Usuario - NovaFin AI Mobile

## Introducción

Bienvenido a NovaFin AI Mobile, su asistente personal de finanzas para Android. Este manual le guiará en el uso de todas las funcionalidades de la aplicación.

## Requisitos Previos

- Dispositivo Android 8.0 (API 26) o superior
- 100 MB de espacio libre en almacenamiento
- Conexión a internet (solo para la primera instalación)

---

## 1. Instalación

### Instalar la APK

1. Copie el archivo `NovaFin-Debug.apk` a su dispositivo Android
2. Abra el archivo desde un explorador de archivos
3. Si solicita permisos, habilite "Orígenes desconocidos"
4. Toque "Instalar"
5. Una vez instalada, toque "Abrir"

### Configuración Inicial

1. Al abrir la app por primera vez, se le pedirá configurar una clave de acceso
2. Ingrese una clave de al menos 4 caracteres
3. Confirme la clave
4. Toque "Iniciar"

---

## 2. Navegación

### Menú Principal (Móvil)

- Toque el ícono ☰ (hamburguesa) en la esquina superior izquierda

- Toque una sección para navegar

### Barra de Navegación (Desktop)

## 3. Dashboard

### Ver el Resumen

1. La app inicia mostrando el Dashboard del mes actual
2. Las tarjetas muestran:
   - **Total Gastado:** Suma de todos los gastos del mes
   - **Ingresos:** Total de ingresos registrados
   - **Balance:** Diferencia entre ingresos y gastos
   - **Deudas Pendientes:** Total de deudas activas
3. El gráfico de torta muestra la distribución por categoría
4. El gráfico de barras compara los últimos meses

### Usar Filtros

1. Toque los filtros en la parte superior
2. Seleccione categoría, estado, o rango de fechas
3. Los gráficos y tarjetas se actualizan automáticamente
4. Toque "Limpiar" para quitar todos los filtros

---

## 4. Registro de Gastos

### Agregar un Gasto

1. Navegue a "Gastos"
2. Toque el botón "+" (agregar)
3. Complete el formulario:
   - **Fecha:** Seleccione la fecha del gasto
   - **Descripción:** Nombre del gasto (ej: "Supermercado")
   - **Categoría:** Seleccione la categoría
   - **Monto:** Ingrese el monto en pesos
4. Toque "Guardar"

### Editar un Gasto

1. Toque el ícono ✏️ junto al gasto
2. Modifique los campos necesarios
3. Toque "Guardar"

### Eliminar un Gasto

1. Toque el ícono 🗑️ junto al gasto
2. Confirme la eliminación

### Cambiar Estado de un Gasto

- **Pendiente:** El gasto aún no se pagó
- **Pagado:** El gasto ya se pagó
- **Cancelado:** El gasto fue cancelado (se excluye de totales)

### Mover Gastos al Próximo Mes

1. Seleccione los gastos con las casillas de verificación
2. Toque "Mover al mes siguiente"
3. Los gastos se transfieren al mes entrante

### Exportar Gastos

1. Aplique los filtros deseados
2. Toque "Exportar CSV"
3. Comparta el archivo generado

---

## 5. Gestión de Deudas

### Registrar una Deuda

1. Navegue a "Deudas"
2. Toque "Agregar deuda"
3. Complete:
   - **Nombre:** Descripción de la deuda
   - **Monto Total:** Cuánto debe en total
   - **Cuotas:** Número de cuotas
   - **Fecha de inicio:** Cuándo empezó
4. Toque "Guardar"

### Registrar un Pago

1. Toque "Registrar pago" en la deuda
2. Ingrese el monto pagado
3. Toque "Guardar"
4. La barra de progreso se actualiza automáticamente

### Ver Progreso

- Cada deuda muestra una barra de progreso
- El porcentaje indica cuánto ha pagado
- Las deudas pagadas se muestran con estado "Pagada"

---

## 6. Presupuestos

### Crear un Presupuesto

1. Navegue a "Presupuestos"
2. Toque "Crear presupuesto"
3. Ingrese el nombre y el monto total
4. Asigne montos por categoría
5. Toque "Guardar"

### Seguir el Presupuesto

- Cada categoría muestra una barra de progreso
- **Verde:** Menos del 80% usado
- **Amarillo:** 80-99% usado
- **Rojo:** Excedido (100% o más)

### Copiar al Próximo Mes

1. Toque "Copiar" en el presupuesto actual
2. El presupuesto se duplica para el mes entrante

---

## 7. Objetivos de Ahorro

### Crear un Objetivo

1. Navegue a "Objetivos"
2. Toque "Crear objetivo"
3. Complete:
   - **Nombre:** Nombre del objetivo
   - **Monto Objetivo:** Cuánto quiere ahorrar
   - **Fecha Límite:** Cuándo quiere lograrlo
   - **Color:** Color del indicador
   - **Icono:** Icono representativo
4. Toque "Guardar"

### Registrar Ahorro

1. Toque "Agregar ahorro" en el objetivo
2. Ingrese el monto ahorrado
3. Toque "Guardar"
4. La barra de progreso se actualiza

### Ver Proyección

- La app calcula si alcanzará la meta a tiempo
- Muestra el porcentaje completado

---

## 8. Configuración

### Acceder

1. Toque el ícono ⚙️ (engranaje) en la esquina superior derecha
2. Se abre el modal de configuración

### Cambiar Clave

1. Ingrese su clave actual
2. Ingrese la nueva clave
3. Confirme la nueva clave
4. Toque "Cambiar clave"

### Eliminar Clave

1. Ingrese su clave actual
2. Toque "Eliminar clave"
3. La app不再 solicitará clave al iniciar

### Exportar Backup

1. Toque "Exportar copia"
2. Se genera un archivo JSON con toda la base de datos
3. Compártalo o guárdelo en un lugar seguro

### Importar Backup

1. Toque "Importar copia"
2. Pegue el contenido JSON de la copia
3. La app se reiniciará con los datos restaurados

---

## 9. Hábitos Financieros

### Crear un Hábito

1. Navegue a "Hábitos"
2. Toque "Crear hábito"
3. Complete:
   - **Nombre:** Nombre del hábito
   - **Descripción:** Descripción del hábito
   - **Frecuencia:** Diario, semanal, mensual
   - **Meta de días:** Cuántos días quiere cumplir
4. Toque "Guardar"

### Registrar Día Completado

1. Toque el hábito
2. Toque "Marcar como completado"
3. La racha se incrementa

### Ver Progreso

- La barra muestra el porcentaje de días completados
- La racha actual se muestra como "🔥 X días"
- La racha máxima se muestra como "Mejor racha: X días"

## 10. Preguntas Frecuentes

### ¿Dónde se guardan mis datos?

Los datos se almacenan localmente en la base de datos SQLite del dispositivo. No se envían a ningún servidor.

### ¿Puedo usar la app sin internet?

Sí, la app funciona completamente offline. Solo necesita internet para la primera instalación.

### ¿Cómo hago una copia de seguridad?

Vaya a Configuración (⚙️) → "Exportar copia". Guarde el archivo JSON en un lugar seguro.

### ¿Qué pasa si olvido mi clave?

Puede eliminar la clave desde Configuración si ya tiene acceso, o reinstalar la app (perderá los datos).

### ¿Puedo importar datos de la versión de escritorio?

La app tiene métodos de importación de backup que son compatibles con el formato JSON de la versión de escritorio.

### ¿La app envía mis datos a internet?

No. Toda la información se mantiene localmente en su dispositivo. No hay servidores externos.

## 11. Solución de Problemas

### La app no se instala

- Verifique que tiene Android 8.0 o superior
- Habilite "Orígenes desconocidos" en Ajustes → Seguridad
- Libere espacio en almacenamiento

### La app se cierra al iniciar

- Reinicie el dispositivo
- Borre los datos de la app desde Ajustes → Aplicaciones
- Reinstale la APK

### Los datos no se guardan

- Verifique que tiene espacio en almacenamiento
- Reinicie la app
- Si el problema persiste, restaure desde un backup

### Las notificaciones no funcionan

- Verifique los permisos de notificación en Ajustes → Aplicaciones → NovaFin AI
- Active las notificaciones
